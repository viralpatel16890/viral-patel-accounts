# viral-patel-accounts

A comprehensive financial management application with Google Sheets integration, budgeting, and transaction tracking capabilities.

## Features

- Dashboard with financial insights and KPIs
- Transaction management and categorization
- Budget planning and tracking
- Google Sheets integration for data synchronization
- Data quality monitoring and validation
- Responsive design with accessibility compliance

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/viralpatel16890/viral-patel-accounts.git
   cd viral-patel-accounts
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Copy `.env.local.example` to `.env.local` (if available)
   - Configure your Google Sheets credentials

### Running the Application

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

```bash
npm run build
```

## Project Structure

- `src/app/` - Main application components and pages
- `src/components/` - Reusable UI components
- `src/services/` - API and data services
- `src/utils/` - Utility functions
- `public/` - Static assets
